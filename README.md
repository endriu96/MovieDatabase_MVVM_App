
# MovieDatabase_MVVM_App
This is the biggest project in my portfolio with over 100 classes inside! Simple database app with movies , artists, upcoming movies and tv series . 

# Used technologies
-LiveData
-MVVM design pattern
-Coroutines
-Retrofit
-Room
-Dagger
-Glider
